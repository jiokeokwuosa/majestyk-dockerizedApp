## Requirements and Installation

To install and run this project you would need to have installed:
- Node Js
- Git
- PHP
- Clone this repo using `git clone https://github.com/jiokeokwuosa/majestyk-dockerizedApp.git`

To run backend:
```
$ cd app-backend
$ composer install
$ php artisan serve
```

To run frontend:
```
$ cd app-frontend
$ npm install
$ npm start
```
